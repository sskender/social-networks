# Node.js backend

### Run

```bash
npm start
```

### Run development

```bash
npm run dev
```

### Linter

```bash
npm run lint
```

```bash
npm run lint:fix
```

## API

- GET artists:
`http://localhost:3000/api/v1/content/artist`

- GET artist details:
`http://localhost:3000/api/v1/content/artist/150642`
