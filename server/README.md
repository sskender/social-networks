# Node.js backend starter

Starter kit for developing backend in Node.js

## What is inside?

- cors
- express
- helmet
- mongoose
- mongoSanitize
- morgan
- xss
- eslint
- nodemon

### Run

```bash
npm start
```

### Run development

```bash
npm run dev
```

### Linter

```bash
npm run lint
```

```bash
npm run lint:fix
```

### Docker

```bash
docker build --tag sskender/app:latest .
```
